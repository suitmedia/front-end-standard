## Use class to style, avoid using tag. Sebisa mungkin hindari memberi style menggunakan tag element. Gunakan class agar style yang dibuat scalable dan lebih mudah di maintain. Selain itu apabila ke depannya markup diubah, artinya kita harus mengubah css kita juga. Dengan menggunakan class, walau markup berubah, tidak akan mempengaruhi style. ```css /* NO! */ .highlight { float: left; width: 30%; margin: 0 3%; h2 { text-transform: capitalized; } ul { color: #444; } } /* GOOD! */ .highlight { float: left; width: 30%; margin: 0 3%; } .highlight-title { text-transform: capitalized; } .highlight-list { color: #444; } ```
