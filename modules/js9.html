## Selalu set event namespace Pada saat meregister sebuah event ke sebuah element, selalu gunakan namespace. Hal ini bertujuan agar developer lebih mudah detach event apabila sebuah element mempunyai event contoh click lebih dari 2. ``` const toggleMenu = () => { // Put magic here } // attach event $btn.on('click.toggleMenu', toggleMenu); // detach event with affecting other click event attached. $btn.off('click.toggleMenu', toggleMenu); ```
