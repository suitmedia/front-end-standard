## Ecmascript Next Suitmedia menggunakan Ecmascript Next (ESNext) dalam membuat Javascript di mana menggunakan [Babel](https://babeljs.io/) untuk compile syntax ESNext menjadi ES5 compatible. Fitur yang wajib diketahui adalah: * Enhanced object literals * Template strings * Destructuring * Rest dan spread operator * Promise Pelajari syntax/fitur ESNext [di sini](https://babeljs.io/learn-es2015/).
