## Don`t change base style Jangan menambahkan/mengubah behavior style yang sudah dimiliki tiap element. Apabila ingin melakukan itu, gunakan class. Hal ini untuk mencegah hal yang tidak diinginkan apabila user meng-input content dari editor. Kalaupun memang ada style khusus untuk tiap-tiap element contoh untuk article, wrap dengan sebuah class lalu declare contextual style berdasarkan wrappernya. ```css // ‚òπÔ∏è h1 { text-decoration: underline; text-transform: uppercase; } textarea { min-height: 90px; -webkit-appearance: none; } ul { width: 400px; } // üòé .heading { text-decoration: underline; text-transform: uppercase; } .field { min-height: 90px; -webkit-appearance: none; } .list { width: 400px; } // Contextual Style .article { h1 { text-transform: uppercase; } p { font-size: 13px; font-weight: 300; } } ```
