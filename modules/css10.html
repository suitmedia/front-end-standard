## Don`t use BEM SCSS shortcut Memang memudahkan jadi tidak perlu menulis berulang-ulang, namun itu membuat file .scss susah untuk di index/search dan akan merepotkan developer penerus. ``` // ☹️ .highlight { ... &__title { ... } &__list { ... } } // 😎 .highlight { ... } .highlight__title { ... } .highlight__list { ... } ```
