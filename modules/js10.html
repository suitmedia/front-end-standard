## Return early Return early apabila element yang akan dipakai tidak ada. Tujuannya adalah: * Body code akan tetap berada di level paling luar * Mengurangi 1 indented/nested code Perlu diperhatikan bahwa cara pengecekan sebuah element ada atau tidak, berbeda antara element dan nodelist/jQuery Objet. ``` // ☹️ (() => { const $btn = $('button'); if ( btn.length ) { btn.click( openMenu ); } })(); // 😎 (() => { const $btn = $('button'); if ( !$btn.length ) return; $btn.click( openMenu ); })(); ```
